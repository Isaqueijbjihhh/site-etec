Aps – Analise e Projetos de sistemas

Resumo:

A disciplina de APS tem como objetivo proporcionar aos alunos a aplicação prática dos conhecimentos adquiridos nas demais disciplinas do curso técnico, por meio do desenvolvimento de projetos, atividades e estudos de casos reais. Durante as aulas, alunos aprendem:

• Planejamento e organização de projetos técnicos, desenvolvendo cronogramas e etapas de trabalho.
• Trabalho em equipe, aprimorando habilidades de comunicação, colaboração e divisão de tarefas.
• Utilização de ferramentas, técnicas e linguagens de programação para implementar soluções práticas.
• Documentação adequada dos projetos, incluindo relatórios, diagramas e apresentações.
• Pesquisa e aplicação de novas tecnologias e boas práticas do mercado.

Objetivo geral:

Capacitar o aluno a integrar teoria e prática, desenvolvendo autonomia, responsabilidade e capacidade de resolver problemas reais na área de desenvolvimento de sistemas. Projetos típicos da disciplina: Desenvolvimento de protótipos de sistemas. Soluções para problemas técnicos específicos. Estudos de caso aplicados a cenários reais. Apresentações e relatórios técnicos detalhados.

PW – Programação Web

Resumo:

A disciplina de PW (Programação Web) tem como objetivo introduzir os alunos ao desenvolvimento de sites e aplicações para a web, desde o front-end (parte visual) até conceitos básicos de back-end.

• HTML: linguagem de marcação usada para estruturar o conteúdo da web (textos, formulários, etc.)
• CSS: linguagem de estilo usada para deixar os sites bonitos e responsivos (cores, fontes.).
• JavaScript (JS): linguagem de programação para adicionar interatividade aos sites (animações, validação de formulários, etc.)
• Bootstrap: framework que facilita a criação de sites modernos com design responsivo (ótimo celular e desktop).

Noções básicas de responsividade, design visual, estruturas de navegação, e boas práticas no desenvolvimento web. Objetivo geral: Capacitar o aluno a construir páginas web bem estruturadas, com aparência profissional, acessíveis e adaptáveis a diferentes dispositivos. Projetos típicos da disciplina: Landing pages (páginas de apresentação). Portfólios pessoais. Mini sistemas de cadastro ou login (parte visual). Utilização de bibliotecas e componentes visuais prontos.

DD-Design Digital

Resumo:

O Design Digital é fundamental para que os alunos entendam como projetar interfaces funcionais e visualmente atrativas. Durante as aulas, os alunos aprendem:

• Fundamentos do design gráfico aplicados à área digital.
• Noções de tipografia, cores, contraste e composição visual.
• Criação de wireframes e protótipos com ferramentas como Figma, Canva ou Adobe XD.
• Princípios de usabilidade e acessibilidade em interfaces digitais.
• Técnicas para tornar sistemas mais intuitivos e amigáveis ao usuário.

Objetivo geral:

Desenvolver a capacidade de criar interfaces bem estruturadas, que proporcionem uma boa experiência ao usuário e valorizem os sistemas desenvolvidos.

Bd-Banco de dados

Resumo:

Essa disciplina ensina como projetar, implementar e gerenciar bancos de dados relacionais. Durante as aulas, os alunos aprendem:

• Modelagem de dados (entidade-relacionamento, DER).
• Projeto de banco de dados normalizado.
• Linguagem SQL (DDL e DML) para manipulação de dados.
• Criação de consultas complexas com JOINs, GROUP BY, HAVING.
• Administração de banco de dados, backup, restore e permissões.
• Introdução a bancos de dados não relacionais (ex: MongoDB).

Objetivo geral:

Habilitar o aluno a desenvolver sistemas que utilizam bancos de dados para armazenar informações de forma estruturada, segura e eficiente.

Fd-Fundamentos da informatica

Resumo:

Disciplina essencial para desenvolver o raciocínio lógico e a base da programação. O aluno aprende a criar algoritmos eficientes, utilizando estruturas de controle e dados. Durante as aulas, os alunos aprendem:

• Lógica de programação e construção de algoritmos passo a passo.
• Estruturas condicionais (if, else, switch) e de repetição (for, while).
• Utilização de variáveis, operadores e tipos de dados.
• Criação de funções, modularização e reutilização de código.
• Estruturas de dados como vetores, matrizes e listas.
• Resolução de problemas com raciocínio lógico e otimização.

Noções básicas de responsividade, design visual, estruturas de navegação, e boas práticas no desenvolvimento web.

Objetivo geral:

Capacitar o aluno a desenvolver soluções computacionais básicas e intermediárias, utilizando lógica estruturada e boas práticas de programação.
