3º ano: Aps – Analise e Projetos de sistemas

Resumo:

Ensinar como planejar, analisar e projetar sistemas de informação de forma estruturada, preparando o aluno para atuar no desenvolvimento de softwares com qualidade e alinhamento às necessidades do cliente.

Matérias do 3º ano de APS

• 1.Coleta de informações com usuários e stakeholders. Técnicas: entrevistas, questionários, observação. Documentação dos requisitos funcionais e não funcionais. Eficiência de algoritmos (complexidade)..

•2.Modelagem de Sistemas Diagramas UML (Unified Modeling Language): Casos de uso. Diagramas de classes. Diagramas de sequência. Diagramas de atividades. Modelos de dados (ER – Entidade-Relacionamento). .

•3. Análise Estruturada e Orientada a Objetos Diferenças entre análise estruturada e OO. Padrões de projeto (Design Patterns) básicos. Definição de classes, atributos e métodos.

• 4.Projeto de Sistemas Projeto arquitetural (camadas: apresentação, negócios, dados). Definição de interfaces. Protótipos de telas e usabilidade.

•5. Ferramentas e Metodologias Ferramentas CASE (Computer-Aided Software Engineering). Metodologias ágeis (Scrum, Kanban). Documentação e padronização.

6. Validação e Verificação Revisão de requisitos e modelos. Testes iniciais de projeto.

Habilidades desenvolvidas Levantar e documentar requisitos. Criar diagramas UML para modelar sistemas. Projetar soluções de software alinhadas ao negócio. Usar ferramentas para apoiar o desenvolvimento. Trabalhar em equipe com metodologia ágil.

8. Projeto Integrador / APS (Atividades Práticas Supervisionadas) Desenvolvimento de um sistema completo em grupo. Aplicação prática de tudo o que foi aprendido. Apresentações e relatórios.

Dicas para se dar bem no 3º ano:

Comece a estudar Git/GitHub: será essencial para trabalho em equipe. Pratique programação fora das aulas (em sites como HackerRank, LeetCode, etc.). Use ferramentas de modelagem como StarUML ou Draw.io. Aprenda a usar um banco de dados de verdade, como PostgreSQL, MySQL ou SQL Server. Se quiser, posso te ajudar a montar um plano de estudos baseado na sua grade específica. Você estuda em qual instituição? Posso procurar a grade curricular exata se me disser o nome e campus..

Projetos típicos da disciplina:
Desenvolvimento de protótipos de sistemas.
Soluções para problemas técnicos específicos.
Estudos de caso aplicados a cenários reais.
Apresentações e relatórios técnicos detalhados.

3º ano: PW – Programação Web

Resumo:

Ensinar a construir aplicações web modernas, com front-end, back-end, banco de dados e integração, utilizando as principais tecnologias da web..

• HTML: linguagem de marcação usada para estruturar o conteúdo da web (textos, formulários, etc.)
• CSS: linguagem de estilo usada para deixar os sites bonitos e responsivos (cores, fontes.).

• 1. Fundamentos da Web Funcionamento da internet e HTTP/HTTPS. Estrutura de páginas web (HTML5, CSS3). Conceitos de cliente e servidor.

• 2. Desenvolvimento Front-End HTML semântico. CSS avançado (Flexbox, Grid, animações). JavaScript básico e manipulação do DOM. Frameworks e bibliotecas (ex: Bootstrap, jQuery).

3. Desenvolvimento Back-End Linguagens comuns: PHP, Node.js, Python (Django, Flask). Servidores web (Apache, Nginx). Conceito de API (RESTful APIs). Manipulação de dados com banco de dados.

4. Banco de Dados Web Integração com bancos relacionais (MySQL, PostgreSQL). Uso de consultas SQL para CRUD (Create, Read, Update, Delete). Modelagem de dados para aplicações web.

5. Programação Assíncrona e AJAX Requisições assíncronas para melhorar UX. Consumo de APIs via JavaScript (fetch, XMLHttpRequest).

3º ano: DD-Design Digital

Resumo:

• 1. Fundamentos do Design Princípios básicos: equilíbrio, contraste, alinhamento, repetição, proximidade. Teoria das cores: harmonia, significado das cores, combinação. Tipografia: escolha e aplicação de fontes para legibilidade e estilo.

• 2. Design de Interfaces (UI) Elementos visuais: botões, menus, ícones, formulários. Layouts responsivos para diferentes dispositivos. Wireframes e prototipagem.

• 3. Experiência do Usuário (UX) Pesquisa de usuários e personas. Jornada do usuário e fluxo de navegação. Testes de usabilidade.

4. Ferramentas de Design Digital Adobe Photoshop, Illustrator. Figma, Adobe XD, Sketch. Ferramentas de prototipagem e colaboração.

5. Design para Web e Mobile Adaptar layouts para telas diversas. Boas práticas de acessibilidade digital. Otimização de imagens e elementos visuais para a web.

Objetivo geral:

Desenvolver a capacidade de criar interfaces bem estruturadas, que proporcionem uma boa experiência ao usuário e valorizem os sistemas desenvolvidos.

3º ano: Bd-Banco de dados

Resumo:

Essa disciplina ensina como projetar, implementar e gerenciar bancos de dados relacionais. Durante as aulas, os alunos aprendem:

• 1. Modelagem de Dados Modelagem entidade-relacionamento (ER). Diagrama ER e normalização (1FN, 2FN, 3FN). Modelagem física e lógica.

• 2. Linguagem SQL Comandos básicos: SELECT, INSERT, UPDATE, DELETE. Comandos avançados: JOIN, subconsultas, funções agregadas. Criação e manipulação de tabelas (CREATE TABLE, ALTER TABLE)..

• 3. Procedures, Triggers e Views Criação e uso de procedures (procedimentos armazenados). Triggers para automatizar ações no banco. Views para simplificar consultas e melhorar segurança.

• 4. Transações e Controle de Concorrência Propriedades ACID. Comandos de controle: BEGIN TRANSACTION, COMMIT, ROLLBACK. Locks e deadlocks.

• 5. Administração de Banco de Dados Gerenciamento de usuários e permissões (GRANT, REVOKE). Backup e recuperação. Monitoramento e otimização de desempenho (índices, planos de execução).

• 6. NoSQL e Bancos Não Relacionais (introdução) Conceitos e tipos (documentos, chave-valor, grafos). Exemplos: MongoDB, Redis, Cassandra.


Objetivo geral:

Habilitar o aluno a desenvolver sistemas que utilizam bancos de dados para armazenar informações de forma estruturada, segura e eficiente.

3º ano: Fd-Fundamentos da informatica

Resumo:

Objetivo: fornecer a base essencial sobre como os computadores funcionam, seus componentes, sistemas operacionais e lógica básica de funcionamento da tecnologia.:

•1. Sistemas Operacionais Avançados Gerenciamento de processos e threads. Sistemas de arquivos e permissões. Virtualização e máquinas virtuais.

• 2. Redes de Computadores Protocolos TCP/IP detalhados. Configuração de redes e dispositivos. Segurança em redes: firewalls, VPNs.
• 3. Segurança da Informação Criptografia básica e certificação digital. Políticas de segurança e práticas de proteção. Ataques comuns e formas de prevenção.

• 4. Computação em Nuvem Conceitos e modelos de serviço (IaaS, PaaS, SaaS). Vantagens e desafios da nuvem. Exemplos de provedores: AWS, Azure, Google Cloud.
• 5. Novas Tecnologias e Tendências Internet das Coisas (IoT). Big Data e Inteligência Artificial (introdução). Blockchain (conceitos básicos).

Noções básicas de responsividade, design visual, estruturas de navegação, e boas práticas no desenvolvimento web.

Objetivo geral:

Capacitar o aluno a desenvolver soluções computacionais básicas e intermediárias, utilizando lógica estruturada e boas práticas de programação.

3º ano: Tpa-Técnicas de Programação e Algoritmos

Resumo:

aprofundar a lógica de programação, trabalhar com estruturas de dados simples, modularização, e começar a resolver problemas mais complexos com algoritmos otimizados.

• 1. Estruturas de Dados Avançadas Listas encadeadas (simples, duplamente encadeadas). Pilhas e filas (estáticas e dinâmicas). Árvores (binárias, busca binária). Grafos (conceitos básicos).

• 2. Algoritmos de Ordenação e Busca Ordenação: Quick Sort, Merge Sort, Heap Sort. Busca: busca binária, busca em árvores.

• 3. Recursão Avançada Algoritmos recursivos complexos. Recursão de cauda. Aplicações práticas.
• 4. Programação Orientada a Objetos (POO) Conceitos: classes, objetos, herança, polimorfismo. Encapsulamento e abstração. Projetos orientados a objetos.

• 5. Análise de Complexidade de Algoritmos Notação Big O. Avaliação de tempo e espaço.
• 6. Algoritmos Gulosos e Divide & Conquer Técnicas para resolução de problemas. Exemplos clássicos.


Objetivo geral:

Capacitar o aluno a desenvolver soluções computacionais básicas e intermediárias, utilizando lógica estruturada e boas práticas de programação.
